# Skill: Brand Identity

## Propósito
Gestionar y evolucionar el sistema de identidad visual de Soluciones Inmobiliarias RD.

## Estado actual
Fase 1 — Blanco y negro únicamente.

## Sistema de diseño
Ver `brand/identity.md` para el sistema completo.

## Para evolucionar la identidad
1. Leer `brand/identity.md`
2. Proponer cambios en `decisions/architecture.md` como nueva decisión
3. Actualizar variables CSS en `web/css/styles.css`
4. Actualizar `brand/identity.md`

## Regla de posicionamiento
La marca es "asesor de financiamiento", no "prestamista". Todo el lenguaje visual debe
reforzar profesionalismo, confianza y bienes raíces — no urgencia financiera.
