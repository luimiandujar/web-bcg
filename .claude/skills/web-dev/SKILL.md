# Skill: Web Development

## Propósito
Crear y mantener el sitio web de Soluciones Inmobiliarias RD.

## Stack
- HTML5 semántico
- CSS3 (variables CSS para el sistema de diseño B&W)
- JS vanilla (sin frameworks)

## Convenciones
- Variables CSS en `:root` para toda la paleta y tipografía
- Mobile-first responsive design
- Sin librerías externas (CDN solo para fuentes Google Fonts)
- Imágenes optimizadas, formato WebP cuando sea posible

## Archivos del sitio
```
web/
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── main.js
└── assets/
    └── images/
```

## Para actualizar el sitio
Lee `web/index.html` y `web/css/styles.css` antes de hacer cambios.
